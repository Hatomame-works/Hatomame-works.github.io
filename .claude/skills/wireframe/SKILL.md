# ワイヤーフレーム作成スキル

`args` で渡された対象ページ・セクションについて、HTMLのワイヤーフレームを1ファイル(インラインCSS、または同階層に簡潔なstyle追加)で作成してください。
対象が指定されていない場合は、何のワイヤーフレームを作るか先にユーザーに確認してください。

作成時は以下のルールを**すべて**守ってください。

## レイアウト

- 読む負担を感じさせない、見てわかるレイアウトにする。
- 洗練された、都会的でミニマルなレイアウトにする。
- 余白とバランスを意識し、ページ全体をうまく活用する。
- フォントは `Noto Sans JP`、やや小さめのサイズにする(本文13〜14px程度)。

## デザイン

- 色は「白・黒・グレー」だけを使う。カラーやブランドカラーは使わない。
- 背景は基本白にする。全面が黒のページは作らない(黒は文字・アイコン・強調枠など最小限の使用にとどめる)。
- 無駄な英語表記、装飾的な線、シャドウは一切つけない。
- 図や画像が必要な部分は、すべてグレーのボックス(`background: #ddd` 程度の単色)で表現する。実際の画像は使わない。
- アイコンは Font Awesome の無料版(`font-awesome` CDN の Free アイコン)を使う。有料アイコンは使わない。

## ライティング(本文・ダミーテキスト含む)

- 1文はやや短くする。1文につき読点(、)を最低1回は使う。
- 1〜2文ごとに段落を分ける。
- 漢字は不自然にならない範囲でひらがなにする(例: 「行う」→「おこなう」)。
- 強調したい名称やセリフには「」を使う。
- 誤字脱字がないか確認し、文体は「です・ます」調に統一する。

## 出力

- 完成したワイヤーフレームのHTMLファイルを作成または更新する。
- どのページ・セクションを作ったか、ファイルパスを最後に簡潔に報告する。
